### 说明文档
> 星座冲突的模态框，封装弹窗公用部分的css,js,html,便于复用

> css由sass编译

#### 数据字典
+ SMB(small model box):小模态框
+ BMB(big model box):大模态框
+ addMask:添加蒙层
+ addPop:添加弹窗到蒙层
+ closeEvent:给×添加关闭弹窗的事件
